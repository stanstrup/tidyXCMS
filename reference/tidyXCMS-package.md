# tidyXCMS: Tidy Interface for XCMS Metabolomics Data Processing

Provides a tidy interface for working with XCMS metabolomics data. This
package offers functions to extract and manipulate XCMS peak detection
and feature grouping results in long-format tibbles, facilitating
downstream analysis and visualization.
